# Relation Properties Calculator (with Closure)

A simple web-based tool that helps users determine the properties of a relation
‚Äî including **reflexive**, **symmetric**, **transitive**, **antisymmetric**, and **closure** computation.

This project is designed for students learning Discrete Mathematics and Computer Science fundamentals.

---

## üîß Features
- Check if a relation is:
  - Reflexive
  - Symmetric
  - Antisymmetric
  - Transitive
- Computes **reflexive closure**
- Computes **symmetric closure**
- Computes **transitive closure**
- Clean HTML, CSS, and JavaScript structure

---

## üìÇ Technologies Used
- **HTML** ‚Äî Structure
- **CSS** ‚Äî Design
- **JavaScript** ‚Äî Logic & Computation

---

## üöÄ How to Use
1. Input a set (example: `{1,2,3}`)
2. Input a relation in pair format (example: `{(1,1),(1,2),(2,3)}`)
3. Click **Calculate**
4. Results will show all identified properties and closures.
